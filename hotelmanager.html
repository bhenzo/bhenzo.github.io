Super Simple Hotel Manager

Un pequeño software que hice de regalo a unos amigos que estrenaban un hotel hace unos cuantos años. Querían un administrador para alquileres "Super simple" ya que los disponibles comerciales eran demasiado complejos. Hecho en Java SE.




Para este programa, utilicé Java SE con Swing y SQLite. Se realizaron pruebas de rendimiento realizando reservaciones aleatorias en todas las habitaciones por 40 años, y el rendimiento no se vió afectado en lo más mínimo.

El elemento visual para indicar los periodos de alquiler, lo realicé de cero extendiendo un JPanel (en ese momento no encontré una librería que se adecuara a mis necesidades). Los periodos en verdes indican que el inquilino ya completó el pago, y los rojos que adeudan.
