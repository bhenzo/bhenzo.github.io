Turismo Móvil

El proyecto que hice en la universidad para la última materia de la carrera A.P.U. Una aplicación Android para que los turistas pudieran conocer la ciudad.

Este proyecto fue desarrollado en conjunto con unas estudiantes de la carrera de Turismo, y consiste en 2 partes. Una aplicación móvil para los turistas, y una aplicación web, para que los proveedores de servicios turísticos de la localidad pudieran ofrecer sus servicios.

El proyecto incluye un sistema de pagos para recomendar proveedores, y tener privilegios extras para ofrecer sus servicios. Se utilizó TODOPAGO como proveedor de cobros por tarjetas de crédito.

Al finalizar el producto, se realizó una presentación junto con las estudiantes de turismo.
